# NotebookAI移动应用规格说明书

> **应用名称**：NotebookAI
> **平台**：iOS + Android（Flutter）
> **版本**：v1.0.0
> **最后更新**：2026-06-07

---

## 1. 产品定位与愿景

一款**随身知识助手**，把 NotebookLM 的核心能力装进口袋。用户上传文档/视频/音频/网页，AI 自动理解、问答、生成播客、画思维导图。

**核心差异点**：
- 真正的**多模态输入**（视频/音频/网页直接解析）
- **离线可用**（本地向量数据库 + 可选本地模型）
- **私密优先**（数据不上云 / 用户完全控制）

---

## 2. 功能规格

### 2.1 核心功能

| 功能 |描述 |
|------|------|
|文档上传 | PDF、DOCX、TXT、MD、CSV、JSON、HTML、XML |
| 视频上传 | MP4/MOV/AVI/MKV → Whisper 转写 → 自动分块 |
| 音频上传 | MP3/WAV/M4A/OGG/FLAC/AAC → Whisper 转写 |
| URL 解析 | 输入 URL → 自动抓取正文 → 入库 |
| 分类管理 | 文件夹（folder）+ 标签（tag）管理 |
| RAG 问答 | 选文档/文件夹，问任何问题，AI 基于内容回答，引用溯源 |
| 多轮对话 | 支持上下文连续对话（默认 6 轮） |
| Podcast 生成 | 选文档 → AI 生成两人对话 Podcast 脚本 → TTS 合成 MP3 |
| 思维导图 | 基于文档内容生成 Mermaid.js 思维导图，支持导出 PNG |
| 费曼学习 | AI 选取关键段落让你复述，AI 评估理解程度，薄弱概念追踪 |
| 自测题生成 | 选择题 / 填空题 / 简答题，自测后对照答案 |
| 知识关联 | 多份文档间发现概念关联，知识图谱可视化 |
| 知识整理 | 自动提取核心概念 / 主题结构 / 关键结论 |

### 2.2 技术架构

```
┌─────────────────────────────────────────┐
│           Flutter App (iOS+Android)     │
├─────────────────────────────────────────┤
│状态层: Riverpod                       │
│  本地存储: sqflite + chroma-lite         │
│  网络: dio (调用现有 Flask 后端) │
└─────────────────────────────────────────┘
              ↕ REST API
┌─────────────────────────────────────────┐
│  现有 Python Flask 后端                  │
│  (app.py 已有全部 API)                   │
└─────────────────────────────────────────┘
```

**离线策略**：
- 文档向量和元数据本地缓存
- 可选接入 Ollama 本地模型

### 2.3 屏幕清单

| 屏幕 | 功能 |
|------|------|
| 首页 | 最近文档 + 快速上传 FAB |
| 文档列表 | 所有文档（按文件夹筛选） |
| 文档详情 | 文档信息 + 全文预览 |
| 上传 | 文件/视频/音频/URL 上传 |
| 问答 | RAG 问答复制，聊天气泡风格 |
| Podcast | 生成脚本 + 播放 +导出 |
| 思维导图 | Mermaid 渲染 + 导出 PNG |
| 费曼学习 | 段落选取 → 用户解释 → AI 评估 |
| 自测 | 题目生成 → 答题 → 对照答案 |
| 设置 | API 配置 / 离线设置 / 深色模式 |

### 2.4底部导航

## 3. UI/UX 设计规范

- **风格**：Notion 感，简洁专业
